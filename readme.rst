###################
WAGW RESELLER PANEL
###################

wagw reseller is a reseller panel that makes it easy for you to sell WhatsApp services provided by wagw

###################
Server Requirements
###################

-  Minimal PHP 7.2
-  IonCube Loader
-  PDO Active
